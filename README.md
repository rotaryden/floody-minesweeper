# Proxx Test

## Run 

on Linux (tested on Ubuntu 22.04):
```
    ./proxx
```

on Windows 
```
    ./proxx.exe
```


## Build: Linux
Prerequisite: golang 1.19 installed

On unix, run script ./build.sh

```
    go build -o build/proxx ./src
```
